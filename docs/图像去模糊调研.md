# Frequency Domain Research Notes

## Purpose

本文档整理近几年将频域信息纳入图像恢复模型的方法，重点关注：

- 图像去模糊任务中如何使用 Fourier / FFT
- 图像恢复任务中如何使用 DWT / Wavelet
- 这些方法与当前项目的关系
- 基于当前实验现象，对后续架构选择的建议

本文档的目标不是做论文综述大全，而是为当前项目的下一步组网提供可执行的判断依据。

## Current Project Context

当前项目已经验证过一条路线：

- 在较深层特征图上做局部 FFT / polar sampling
- 提取方向先验
- 再驱动 router / experts 做局部恢复

从现有 probe 和训练结果看，这条路线存在明显问题：

- 训练输入只有 `256x256` crop
- 深层特征经过多次下采样后空间分辨率很低
- 再继续切小 patch 做频域统计，局部空间支持不足
- 手工定义的方向先验、置信度和专家映射都不够稳定

因此后续需要重新评估：频域信息到底应该如何进入模型。

## Recent Fourier-Based Methods

### 1. Frequency-domain attention / feed-forward

代表工作：

- FFTformer, CVPR 2023  
  <https://openaccess.thecvf.com/content/CVPR2023/html/Kong_Efficient_Frequency_Domain-Based_Transformers_for_High-Quality_Image_Deblurring_CVPR_2023_paper.html>

核心思路：

- 把特征送入频域
- 在频域中做更高效的 attention 近似
- 在 FFN 中引入频率相关建模
- 重点用在恢复阶段，尤其 decoder

启示：

- Fourier 更常被用来做全局建模和高低频重加权
- 不一定直接拿来估计显式方向角

### 2. Amplitude / phase disentanglement

代表工作：

- Fourier Prior-Based Two-Stage Architecture, CVPRW 2024  
  <https://openaccess.thecvf.com/content/CVPR2024W/NTIRE/html/Nehete_Fourier_Prior-Based_Two-Stage_Architecture_for_Image_Restoration_CVPRW_2024_paper.html>

核心思路：

- amplitude 更多携带退化信息
- phase 更多保留结构信息
- 分阶段修复 amplitude 和 phase

启示：

- 频域不是一个整体
- amplitude / phase 可能承担不同角色
- 只使用 magnitude 很可能会丢失关键结构信号

### 3. Frequency-domain supervision

代表工作：

- FDL, CVPR 2024  
  <https://openaccess.thecvf.com/content/CVPR2024/html/Ni_Misalignment-Robust_Frequency_Distribution_Loss_for_Image_Transformation_CVPR_2024_paper.html>

核心思路：

- 不一定改主干结构
- 直接在频域上设计损失
- 对 amplitude / phase 分布施加约束

启示：

- 频域信息不一定非要做成网络分支
- 也可以作为训练监督

### 4. Fourier as a restoration variable

代表工作：

- FrDiff, ICCV 2025  
  <https://openaccess.thecvf.com/content/ICCV2025/html/Liu_Frequency_Domain-Based_Diffusion_Model_for_Unpaired_Image_Dehazing_ICCV_2025_paper.html>

核心思路：

- 直接在频域生成或校正 amplitude
- 再做 phase correction

启示：

- 更近期的趋势是把频域当作主恢复变量之一
- 而不是只做一个辅助统计

### 5. More expressive Fourier variants

代表工作：

- F2former, WACV 2025  
  <https://openaccess.thecvf.com/content/WACV2025/html/Paul_F2former_When_Fractional_Fourier_Meets_Deep_Wiener_Deconvolution_and_Selective_WACV_2025_paper.html>

核心思路：

- 认为普通 FT 对非平稳信号不够灵活
- 引入 Fractional Fourier Transform
- 结合 selective frequency modeling

启示：

- 如果退化是空间变化且非平稳的，单一全局 Fourier 表示可能不够

## Recent Wavelet-Based Methods

### 1. Wavelet as backbone downsampling / upsampling

代表工作：

- MWCNN, CVPRW 2018  
  <https://openaccess.thecvf.com/content_cvpr_2018_workshops/html/Liu_Multi-Level_Wavelet-CNN_for_Image_Restoration.html>

核心思路：

- 用 DWT 替代普通池化或 stride downsample
- 用 IWT 做重建
- 同时保留多尺度频率信息与空间结构

启示：

- Wavelet 天然适合图像恢复中的 encoder-decoder 框架
- 它不是后处理技巧，而可以直接进入 backbone

### 2. Wavelet for deblurring refinement

代表工作：

- SDWNet, ICCVW 2021  
  <https://openaccess.thecvf.com/content/ICCV2021W/AIM/html/Zou_SDWNet_A_Straight_Dilated_Network_With_Wavelet_Transformation_for_Image_ICCVW_2021_paper.html>

核心思路：

- 保持较高分辨率
- 用 wavelet module 促进高频细节恢复

启示：

- Wavelet 很适合用于高频补偿和细节恢复

### 3. Learnable DWT for blind motion deblurring

代表工作：

- MLWNet, CVPR 2024  
  <https://openaccess.thecvf.com/content/CVPR2024/html/Gao_Efficient_Multi-scale_Network_with_Learnable_Discrete_Wavelet_Transform_for_Blind_CVPR_2024_paper.html>

核心思路：

- 在盲运动去模糊中使用可学习 DWT
- 多尺度建模模糊频率特征
- 关注频率子带中的退化线索

启示：

- 对当前项目最有参考价值
- 证明 wavelet 直接用于 motion deblurring 是成立的

### 4. Low / high frequency decoupling

代表工作：

- WaveFace, CVPR 2024  
  <https://openaccess.thecvf.com/content/CVPR2024/html/Miao_WaveFace_Authentic_Face_Restoration_with_Efficient_Frequency_Recovery_CVPR_2024_paper.html>

核心思路：

- 低频与高频分开恢复
- 不同频率成分用不同子网络处理

启示：

- Wavelet 最自然的优势之一是“频率成分可分治”

## Hybrid Wavelet + Fourier Methods

代表工作：

- WF-Diff, CVPR 2024  
  <https://openaccess.thecvf.com/content/CVPR2024/html/Zhao_Wavelet-based_Fourier_Information_Interaction_with_Frequency_Diffusion_Adjustment_for_Underwater_CVPR_2024_paper.html>

核心思路：

- 先在 wavelet 子带空间中组织局部频率表示
- 再引入 Fourier interaction 做更强的频域交互

启示：

- Wavelet 更适合提供局部空间对齐的频率表示
- Fourier 更适合做全局频谱交互
- 两者不是互斥关系

## Common Design Patterns

从这些工作抽象出来，频域进入图像模型通常有以下几种方式。

### Pattern 1: Frequency-aware backbone

- 在 block 中显式做 FFT / frequency attention
- 或在 backbone 中用 DWT / IWT 替代普通上下采样

### Pattern 2: Frequency prior decomposition

- 分离 amplitude / phase
- 分离 low / high frequency
- 分别恢复后融合

### Pattern 3: Frequency supervision

- 主干不一定频域化
- 但训练中加入频域损失或频率一致性约束

### Pattern 4: Frequency-guided modulation

- 频域分支不直接恢复图像
- 只输出 prior / attention / modulation signal
- 用于调制主干恢复特征

对当前项目最相关的是：

- Pattern 1 中的 wavelet backbone
- Pattern 4 中的 high-resolution learned prior branch

## What Recent Work Suggests for This Project

### What seems weak in the current project direction

现有 polar 思路的核心问题不是“频域无用”，而是：

- 在深层低分辨率特征上继续切小 patch
- 用 hand-crafted FFT magnitude / polar statistics
- 试图直接估计精细角度或 blur type

这条链过长，而且最脆弱的地方是：

- patch 空间支持不足
- magnitude 容易受内容纹理影响
- 方向 / 类型先验与真实退化语义对齐不强

### What seems more promising

结合文献与当前 probe，当前更有希望的方向是：

#### Direction A: Wavelet Blur-Type Prior Branch

- 在输入图像或浅层高分辨率特征上做 1~2 层 DWT
- 得到 `LL/LH/HL/HH`
- 用小 CNN 融合子带特征
- 输出 `B x 4 x H' x W'` 的 blur-type logits / map

这条线的优点：

- 不需要在深层特征上再切很小 patch
- wavelet 子带本身保留空间位置
- 更适合输出局部 map
- 与当前 `short/long x iso/aniso` 的专家定义直接对齐

#### Direction B: Prior-as-modulation, not prior-as-hard-routing

当前实验说明，专家路由容易成为主要不稳定源。更稳的做法是：

- 先保留高分辨率 prior branch
- 但先不用它做 hard expert routing
- 只把它作为主干的 modulation / conditioning signal

例如：

- spatial gate
- FiLM-style affine modulation
- prior-guided decoder attention

### Direction C: Wavelet before Fourier, not Fourier on tiny deep patches

如果还希望保留 Fourier flavor，更合理的是：

- 先用 wavelet 做局部多尺度分解
- 再在 wavelet feature 上做 Fourier interaction

而不是：

- 直接在很小的深层 patch 上做 hand-crafted polar / FFT statistics

## Recommended Next-Step Architecture Direction

基于当前调研与已有实验，建议优先考虑如下方向：

### Preferred route

Wavelet Blur-Type Prior Branch + Strong Single Backbone

具体思路：

1. 输入模糊图或浅层 feature
2. 做 1~2 层 DWT
3. 小 CNN 融合 wavelet subbands
4. 输出 `blur-type map` 或 prior feature map
5. 在 bottleneck / decoder3 调制主干特征
6. 先不强依赖 experts / router

### Why this route is preferred

- 比当前 deep small-patch polar 更稳
- 和最新 wavelet restoration literature 更一致
- 更容易做可视化与定量验证
- 更适合当前 `256x256 crop` 训练设定

## Practical Takeaways

### Things worth keeping

- 关注退化先验本身，而不只是盲目堆 backbone
- 保留多尺度恢复框架
- 继续在 bottleneck / decoder 恢复阶段注入先验，而不是早期 encoder 全网乱插

### Things worth reducing

- 不再继续依赖 hand-crafted polar angle prior
- 不再在深层小 patch 上做精细角度统计
- 不把所有 hope 都压在 4-expert hard routing 上

### Things worth trying next

1. 写一个独立的 wavelet pair probe
2. 验证从 GoPro blur/sharp pair 中构造的伪真值与 wavelet prior 是否相关
3. 如果相关性成立，再做 learned wavelet prior branch
4. 优先先做 modulation，再决定是否要接 router / experts

## Summary

近几年将频域纳入图像模型，主流并不是“在小 patch 上估一个频域角度”，而是：

- 用 Fourier 做频域 attention、幅相分离、频域监督
- 用 Wavelet 做多尺度局部频率分解、低高频分治、backbone 替换
- 用 Wavelet + Fourier 做联合局部/全局频率建模

对当前项目来说，最值得转向的路线不是继续强化 deep small-patch polar，而是：

**基于 wavelet 的高分辨率 learned blur prior / feature modulation 路线。**
